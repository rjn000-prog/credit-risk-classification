# Credit Risk Classification System

## 📌 Project Description
This project predicts whether a customer is High Risk or Low Risk using Machine Learning.

## 📊 Dataset
German Credit Dataset (public dataset)

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Flask
- HTML/CSS

## 🤖 Models Used
- Logistic Regression
- KNN
- Decision Tree
- Random Forest (Best Model)

## 🚀 Features
- Data preprocessing & EDA
- Model comparison & optimization
- Web-based prediction system
